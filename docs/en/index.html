Write Locks

Prevents concurrent execution of specific tasks, ensuring they run sequentially. Mutually exclusive with read locks.

Read Locks

Allows multiple read operations to run in parallel. Mutually exclusive with write locks.

Abortable

Supports AbortSignal to cancel pending tasks waiting for a lock.
